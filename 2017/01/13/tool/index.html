介绍

轻量级标记语言,”使用易读易写的纯文本格式编写文档”


Markdown 的优点


  • 纯文本,所以兼容性极强,可以用所有文本编辑器打开

  • 专注于文字写作而不是排版

  • 格式转换方便,可以轻松转换为 html,ppt, 电子书等

  • Markdown 标记语法简单好记,可读性强

基本语法

标题


  1. 标题前后要留有空行

  2. 英文和中文之间最好留有一个空格

  3. 标题之间可以用”—“形成的横线进行分割


段落


  1. 段落前后不要有别的内容 ,留有空行

  2. 段落不会换行

  3. 段落之后两个空格强制单个段落内容换行

  4. 两端文本之间保留换行就是两个段落了


粗体

使用两个”*”包裹要加粗的文本;


斜体

使用两个””包裹来确定倾斜文字;

无论是有序列表还是无序列表,如果就写一些一级列表,没有什么嵌套就紧凑的写到一起就即可;
如果有多层嵌套,就可以在列表项与列表项之间加一个空行来保持格式

Markdown转html原理

原理:通过正则表达式将md字符串转换为html字符串

自定义实现md文件转html


  1. 下载md转html的 marked.min.js 库文件

  2. 将 marked.min.js 文件引入到 html 文件中

  3. 利用 marked 全局变量进行转换

  4. marked.min.js 不支持代码高亮,可以下载 lightHight.js文件来解决
